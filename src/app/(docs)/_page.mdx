# Nexy : Le Framework Python Qui Ne Promet Pas La Lune 🚀
> (Mais qui vous emmène quand même assez loin !)

## Le Monde Merveilleux des Frameworks Python 🎭

Vous connaissez la chanson : 
- **Django** : Le papa ours qui veut tout contrôler 🐻
- **Flask** : Le bébé ours qui vous laisse faire tout le boulot 🐻‍❄️
- **FastAPI** : L'ado rebelle qui court très vite... mais dans tous les sens 🏃‍♂️

Et nous voilà avec **Nexy** : L'ours qui a lu Marie Kondo et qui aime ranger ! 📚✨

## Ce Que Nexy Est Vraiment 🎯

Non, Nexy ne va pas :
- Faire votre café ☕ (désolé)
- Remplacer tout votre JavaScript (on n'est pas fous !)
- Résoudre la complexité inhérente du développement web 

Mais il va rendre votre vie de dev Python **significativement moins chaotique** !

### La Magie de la Convention (Avec Modération 🧙‍♂️)

```controller.py:python
# app/users/controller.py
@customResponse(type=HTMLResponse)
async def GET():
    return {"users": get_users(), "magicLevel": "modéré"}

# Oui, c'est tout. 
# Non, ce n'est pas de la magie noire.
# Juste de la convention intelligente ! 
```

### Les Actions Serveur (Avec JavaScript Quand Même 😏)

```html 
<!-- Bon d'accord, on a toujours besoin de JavaScript... -->
<!-- Mais au moins on l'écrit moins souvent ! -->
<button action="delete_user" data-id="{{user.id}}">
    Supprimer (et oui, du JS va s'exécuter 🎭)
</button>
```

## L'Architecture Qui Ne Vous Fera Pas Perdre Vos Cheveux 🦱

```plaintext
app/
 ├── controller.py     # La logique (ici !)
 ├── view.html        # L'interface (là !)
 ├── layout.html      # La structure (par là !)
 ├── users/
     ├── controller.py # Vous voyez le pattern ? 😉
     ├── view.html    # Franchement, c'est pas beau ?
```

## Pour Qui Est Nexy ? 🎯

- **Les Fans de Django** : Qui en ont marre de taper `python manage.py [insérer_commande_mystérieuse]`
- **Les Aficionados de Flask** : Qui veulent enfin arrêter de réinventer la roue (carrée)
- **Les Amoureux de FastAPI** : Qui aimeraient bien savoir où mettre leurs fichiers
- **Les Nouveaux** : Qui se demandent pourquoi le web en Python doit être si compliqué

## La Vérité Sur Les Performances 🏎️

Basé sur FastAPI, donc :
- Rapide ? ✅ 
- Très rapide ? ✅
- Plus rapide que PHP ? Probablement (désolé PHP 😅)
- Plus rapide que votre café du matin ? À vous de juger !

## Démarrez Plus Vite Qu'une Installation de Node Modules ⚡

```bash
pip install nexy
nexy new mon-projet
cd mon-projet
nexy dev

# Pendant ce temps-là, npm est toujours en train d'installer...
```

- Et surtout : Python qui reste Python ! 🐍
