# Introduction

Dans l'écosystème du développement moderne, un framework efficace doit répondre à un défi majeur : 
<span className='text-[#A9FFEA]'>comment accélérer le développement tout en maintenant des standards de qualité élevés ?</span>

<Badge>Nexy</Badge> répond à cette problématique en combinant la puissance de FastAPI avec une approche simplifiée du développement web. Notre framework repose sur deux piliers essentiels : **des performances exceptionnelles** et **une expérience développeur optimisée**.

___

## Qu'est-ce que Nexy ?

<Badge>Nexy</Badge> est un framework web moderne basé sur <a href="https://fastapi.tiangolo.com/" className='text-yellow-300 italic' target="_blank">FastAPI</a>, conçu pour démocratiser le développement d'applications web hautes performances.

### Pourquoi Nexy ?

<Mark>
- <Badge>Rapidité</Badge> de développement : Une architecture intuitive pour lancer vos projets efficacement
- <Badge>Performances</Badge> natives : L'excellence technique de FastAPI, enrichie d'optimisations avancées
- <Badge>Simplicité</Badge> d'usage : Une expérience développeur fluide et accessible
- <Badge>Flexibilité</Badge> : Une adaptabilité totale, du projet personnel aux architectures complexes
</Mark>
Que vous construisiez une API REST, une application web complète ou un microservice, <Badge>Nexy</Badge> vous accompagne avec des outils conçus pour maximiser votre productivité.

___

## Avant de commencer

Pour une expérience optimale avec cette documentation, nous recommandons de maîtriser :

- <span className="text-amber-50/50 text-sm ">① </span> **Python** (niveau intermédiaire) - Syntaxe, programmation orientée objet
- <span className="text-amber-50/50 text-sm ">② </span> **FastAPI** (notions de base) - Concepts des API REST et décorateurs Python

> **Note importante :** Débutant avec <a href="https://fastapi.tiangolo.com/" className='text-yellow-300 italic' target="_blank">FastAPI</a> ? Notre documentation détaillée vous guidera à travers chaque concept essentiel.
