
# Introduction

L'objectif fondamental d'un framework est d'optimiser et de rationaliser le processus de développement. 
C'est un ensemble d'outils sophistiqués conçu pour faciliter la création d'applications web performantes 
tout en maximisant la productivité des développeurs.

<Badge>Nexy</Badge> illustre parfaitement cette vision en se concentrant 
sur deux piliers fondamentaux : l'excellence des performances de vos applications et une expérience de 
développement intuitive et enrichissante.
___

## Qu'est-ce que Nexy ?


<Badge>Nexy</Badge> basé sur <a href="https://fastapi.tiangolo.com/" className='text-yellow-300 italic' target="_blank">FastAPI</a> a été pensé pour être simple et facile à utiliser.
Il est conçu pour être utilisé par les développeurs  qui souhaitent créer 
des applications web rapidement et efficacement sans sacrifier les performances.

Que vous soyez un développeur individuel ou que vous fassiez partie d'une équipe plus large, <Badge>Nexy</Badge> peut vous aider à créer des applications Python rapide.
