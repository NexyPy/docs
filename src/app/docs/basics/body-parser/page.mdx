# Analyseur de Corps de Requête

Nexy fournit une analyse complète du corps des requêtes grâce à <Badge>FastAPI</Badge>. Contrairement aux frameworks traditionnels qui nécessitent une configuration manuelle des analyseurs, Nexy gère intelligemment le contenu des requêtes en fonction des types MIME et de leur structure.

<Mark>
  Nexy analyse automatiquement le JSON, les données de formulaire, les fichiers et même les structures imbriquées complexes. Il suffit de définir votre schéma attendu !
</Mark>

---

## Types d'Analyseurs avec Exemples

</CodeCard>

<CodeCard fileName="examples/basic/float.py">
<Code language="python" theme="catppuccin-mocha"
