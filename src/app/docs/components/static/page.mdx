# Les fichiers statiques

Les fichiers statiques sont des éléments immuables essentiels à votre application. Ces ressources - qu'il s'agisse d'images, de feuilles de style, de scripts JavaScript ou de fichiers de police - restent constantes tout au long de l'exécution de votre application.

## Configuration

Votre application dispose d'un espace dédié dans le dossier <Badge>public</Badge>, situé à la racine, pour stocker ces fichiers statiques. Pour une configuration avancée, le fichier <Badge>nexyconfig.py</Badge> vous permet d'ajuster ces paramètres selon vos besoins.

## Intégration

Dans un fichier HTML

<CodeCard fileName="app/View.html">

`<link rel="stylesheet" href="/public/css/style.css">
<script src="/public/js/app.js"></script>
<img src="/public/images/logo.png" alt="Logo">`}
/>
</CodeCard>
